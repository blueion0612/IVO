# IVO System Pipeline - Technical Documentation

## 1. System Overview

IVO (IMU-Vision Overlay) is a contactless presentation control system that utilizes smartwatch IMU sensors and webcam-based hand tracking. The system consists of **6 main pipelines**:

1. **IMU Gesture Recognition Pipeline** - Wrist gesture recognition
2. **Hand Tracking Pipeline** - Webcam-based hand tracking and pointer/drawing
3. **Speech-to-Text Pipeline** - Voice recognition and transcription
4. **Q&A Summarization Pipeline** - Question and answer summarization
5. **OCR & Calculation Pipeline** - Handwriting recognition and formula calculation
6. **Sticky Note Mode Pipeline** - Voice sticky notes and dictionary lookup

## 8. Electron Overlay Architecture

### 8.1 Electron Window Configuration

IVO utilizes Electron's **transparent overlay window** feature to display graphical elements on top of presentations.

```javascript
// BrowserWindow creation settings
const win = new BrowserWindow({
    width: screenWidth,
    height: screenHeight,
    x: 0,
    y: 0,
    transparent: true,           // Transparent background
    frame: false,                // Frameless (no title bar)
    alwaysOnTop: true,           // Always on top
    skipTaskbar: true,           // Hidden from taskbar
    hasShadow: false,            // No shadow
    focusable: false,            // Non-focusable (keep PPT focus)
    webPreferences: {
        nodeIntegration: false,
        contextIsolation: true,
        preload: preloadPath      // IPC bridge
    }
});

// Mouse event pass-through configuration
win.setIgnoreMouseEvents(true, { forward: true });
```
